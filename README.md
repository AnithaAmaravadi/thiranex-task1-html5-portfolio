# Personal Portfolio Website

A clean, responsive personal portfolio website built using semantic HTML5 and custom CSS3. Designed to showcase projects, skills, and contact information with a modern, user-friendly layout.

## 🛠️ Features
* **Semantic HTML5:** Built using standard web structures (`<nav>`, `<section>`, `<footer?>`, etc.).
* **Responsive Styling:** Clean layout that automatically adapts to different screen sizes.
* **Navigation:** Multi-page setup covering Home, About, Projects, and Contact details.
* **Interactive Forms:** Standardized user input fields for seamless contact inquiries.

## 📂 Project Structure
* `index.html` - Home / Landing Page
* `about.html` - About Me Page
* `projects.html` - Works / Projects Page
* `contact.html` - Contact Form Page
* `style.css` - Custom styling rules
