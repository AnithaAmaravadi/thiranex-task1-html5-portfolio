# HTML5 Semantic Portfolio Website

A multi-page personal portfolio website developed as part of the **Thiranex Web Development Internship - Task 1**. This project demonstrates the implementation of semantic HTML5 structure, accessibility best practices (WCAG), and SEO-friendly web development techniques.

## 🛠️ Features

* **Semantic HTML5 Structure:** Built using semantic elements such as `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`.
* **Accessibility Focused:** Includes ARIA labels, proper form labeling, and keyboard-friendly navigation.
* **SEO Friendly:** Optimized with essential meta tags for better search engine visibility.
* **Multi-Page Website:** Separate pages for Home, About, Projects, and Contact.
* **Accessible Contact Form:** User-friendly form with proper labels and required field validation.

## 📂 Project Structure

* `index.html` - Home Page
* `about.html` - About Me Page
* `projects.html` - Projects Showcase
* `contact.html` - Contact Form
* `style.css` - Website Styling

## 🚀 Technologies Used

* HTML5
* CSS3

Build a fully accessible and semantic portfolio website adhering to modern HTML5 standards and WCAG accessibility guidelines.
